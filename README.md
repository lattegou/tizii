## 简介

AirTiz 是一款为 macOS 打造的轻量 VPN ，基于苹果原生 **Swift** 与内核 **mihomo** 构建。高频操作收敛到托盘内，菜单栏实时网速抬眼即知。

## 特性

🍎 **原生 Mac UI** — 采用 Mac 原生的 Swift 技术栈，轻量、高性能、高颜值。

🎯 **托盘快捷操作** — 代理开关、模式与节点选择等高频操作在托盘快捷实现，专注你的工作流。

📊 **实时网速监控** — 菜单栏直观展示上下行网速与分流模式，可单纯作为网速监控工具使用。

🔒 **安全隐私优先** — 架构加密通信，不追踪任何用户，绝不访问用户网络流量数据。


## 系统要求

macOS 14.0 及以上版本

Apple Silicon（Intel 版本敬请期待）

AirTiz 是一款**图形界面客户端**，代理功能由开源内核 mihomo 提供。AirTiz 本身**不提供 VPN 服务器或代理节点**，您需要自行配置订阅后使用。
